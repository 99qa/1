# YouSound!（ver 1.1.0）

## 概要

YouTubeのプレイリストURLを入力するだけで、簡単に音楽を楽しめるウェブアプリケーションです。

面倒なアカウント登録や追加料金は一切不要で、完全無料でご利用いただけます。

## 使用方法

1.YouTubeのプレイリストURLをコピーします。

2.YouSound!に貼り付けます。

3.「読み込み」ボタンをタップします。

## スマホでバックグラウンド再生する方法

1.動画を再生します。

2.ホーム画面に戻ります。

3.コントロールセンターを開きます。

4.動画を再生します。
